# Calculate Distance Between Points.
Java program to calculate distance between two points. This Program uses interface for abstraction.

## How to Run
### Dependencies

```
openjdk 11.0.1+
```
### Clone the Repo
```
$ git clone https://github.com/anxmukul/java-interface.git
```
### How to run

```
$ cd java-interface
$ javac Main.java
$ java Main
```
